简介

Based on my understanding of OA process, this process is specially used for approval of various document flow!

This process tool is simple to use and easy to understand. The main style is that the process document and the underlying document are separate and highly decoupled...

Process definition, enough odoo

Under the configuration menu, the process defines the function, creates the process, obtains the different process, runs the process engine in the program, and looks for the most appropriate approver. The specific operation is as follows:

Main function configuration:

  1. Document type configuration
  2. The process definition
  3. Employee organization configuration
  4. Process delegation configuration (configurable on demand)

Important Note

The configured fields are all prompted. Follow the prompts
